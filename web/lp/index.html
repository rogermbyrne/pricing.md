> I need transactional emails for user notifications
>
Comparing Email...
Resend Free: 3K/mo (100/day)
Resend Pro: $20/mo (50K)
Free tier caps at 100/day. Pro: 50K emails for $20/mo. Uses standard SMTP — switch providers anytime.

Descope: 7.5K MAUs free

40x difference in free tier. Auth0: 25K free then $35/mo. WorkOS: 1M free then $2,500/1M overage.
> database for a side project
>
Comparing Database...
Neon Free: 0.5GB, PostgreSQL
Turso Free: 5GB, SQLite
Neon: drop-in PostgreSQL, switch to any Postgres host anytime. Turso: 10x more storage but SQLite dialect.
> ci/cd pipeline — circleci vs buildkite
>
Comparing CI/CD...
CircleCI: $15/seat/mo
Buildkite: $30/seat/mo
CircleCI: cheaper per-seat + $15/25K credits. Buildkite: 2x per-seat but includes hosted runners and 10 agents.
